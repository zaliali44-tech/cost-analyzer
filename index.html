Formula Glossary

This section stays at the bottom so users can understand how each metric is calculated.
Volume-Based HC Required HC = (Volume × AHT) / 3600 / Hours / Occupancy Adjusted Required HC = Required HC / (1 - Shrinkage) Daily Production Billing Required HC = Required Production Hours Per Day / Production Hours Per Agent Per Day Monthly Production Billing Required HC = Required Production Hours Per Month / Production Hours Per Agent Per Month Common Formulas Staff Variance = Available HC - Required HC Leakage = max(Available HC - Required HC, 0) × Salary Per Employee Revenue Risk = max(Required HC - Available HC, 0) × (Revenue / Required HC) Margin % = ((Revenue - Actual Cost) / Revenue) × 100 Optimized Margin % = ((Revenue - Optimized Cost) / Revenue) × 100
