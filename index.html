Formula Glossary

This section stays at the bottom so users can understand how each metric is calculated.
Volume-Based HC Required HC = (Volume × AHT) / 3600 / Hours / Occupancy Adjusted Required HC = Required HC / (1 - Shrinkage) Daily Production Billing Required HC = Required Production Hours Per Day / Production Hours Per Agent Per Day Monthly Production Billing Required HC = Required Production Hours Per Month / Production Hours Per Agent Per Month Commercial Logic If Cost Input Type = Salary per Employee: Current Cost = Available HC × Salary per Employee Optimized Cost = Required HC × Salary per Employee If Cost Input Type = Total Cost: Cost per HC = Total Cost / Available HC Current Cost = Total Cost Optimized Cost = Cost per HC × Required HC Profit = Revenue - Current Cost Leakage = max(Available HC - Required HC, 0) × Cost per HC Revenue Risk = max(Required HC - Available HC, 0) × (Revenue / Required HC) Margin % = ((Revenue - Current Cost) / Revenue) × 100 Optimized Margin % = ((Revenue - Optimized Cost) / Revenue) × 100 Staff Variance = Available HC - Required HC
